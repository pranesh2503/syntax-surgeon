# 🔬 The Syntax Surgeon
AI-Powered Code Fixer & Conversational Assistant using IBM Granite 3.3 2B

## 📌 Description
The Syntax Surgeon is an intelligent chatbot that fixes Python code errors,
formats code using Black/autopep8, and holds natural conversations —
all powered by IBM Granite 3.3 2B running locally without any API keys.

## 🚀 Run in Google Colab (One Click)
[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/pranesh2503/syntax-surgeon/blob/main/syntax_surgeon.ipynb)

## ✨ Features
- 💬 Natural language conversation
- 🔧 Automatic Python code fixing
- ✅ Syntax validation using AST
- 📄 Export chat as TXT or PDF
- 🔗 Share chat via Pastebin

## 🛠️ Tech Stack
- IBM Granite 3.3 2B Instruct
- Gradio 4.0+
- HuggingFace Transformers
- Black + autopep8
- ReportLab
- Google Colab (GPU)
